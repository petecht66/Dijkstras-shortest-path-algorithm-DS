# Dijkstras-shortest-path-algorithm-DS
Date Submitted: December 10, 2024
Last Updated on GitHub: April 4, 2026

This repository contains my complete work for the Dijkstra's algorithm project of CISC220: Data Structures at the University of Delaware College of Engineering Department of Computer and Information Sciences. I completed this course in the Fall of 2024, with Professor Leighanne Hsu being the course instructor. The purpose of this course was to introduce students to topics such as data type abstraction, recursion, arrays, stacks, linked lists, dynamic storage management, trees, graphs, tables, sorting, searching, and more.

This project contained work with Dijkstra's shortest path algorithm. Some of the code was provided by Professor Hsu. I created the following methods for this assignment: runDijkstra, setDistances, minDistance, and printPath. Then, in the main file, I tested these methods on the maps that were provided by Professor Hsu.

Two folders were created for this repository. The 'maps' folder contains the three maps that were used in this project. The 'programming-files' folder contains my work in C++.
